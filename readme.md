# Dependencies

* Ruby 2.0.0-p0
* node: `$ brew install node`
* Mocha: `$ sudo npm install -g mocha`
* Chai `$ sudo npm install -g chai`
* Coffeescript: `$ sudo npm install -g coffee-script`

Run `$ bundle install`

***

# Coffeescript

* Run `$ guard`

***

# Testing

* Run `npm test`

***

# Server

* Run `npm start`
* Go to http://localhost:3000
